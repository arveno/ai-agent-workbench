# 配置弹窗 UI 验收清单

本清单用于验收四个配置弹窗完成 shadcn/ui 迁移后的视觉一致性、交互状态、滚动行为和主流程回归。

检查范围：

- 模型配置弹窗
- 数据源配置弹窗
- 工具库配置弹窗
- 工作流 / Prompt 配置弹窗

说明：

- `[x] 已检查` 表示已通过代码结构、样式约束或工程命令确认。
- `[ ] 待人工确认` 表示需要在浏览器中打开页面进行视觉或交互确认。
- 本文档不包含任何 API Key、数据库连接串或其他 secret。
